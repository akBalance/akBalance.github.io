下载MYSQL

https://dev.mysql.com/downloads/installer/ 去这个网址下载MYSQL, 默认安装就行，里面包含了Workbench管理工具

授权

在安装好后的MYSQL命令窗口依次执行下面三条语句

数据库表结构

user表

字段名 类型 说明
id int 用户id，不为空，主键，自增长
administrator varchar 所在单位，不为空
name varchar 用户名称，不为空
phone varchar 用户电话

water表

字段名 类型 说明
id int 用水类型id，主码，不为空
type varchar 用水类型名称，不为空
price double 用水类型的价格

uw表

字段名 类型 说明
usrid int 用户表的id，不为空
waterid int 用水类型的id，不为空
usedate varchar 用户用水的时间，不为空
mydate varchar 录入数据的时间，不为空
start double 用水的起始值，不为空
end double 用水的结束值，不为空
money double 水费，不为空

workbench中使用触发器

  1. 鼠标右键选择要建立触发器的表,选择Alter Table

  1. 点击下方的trigger按钮

  2. 选择触发器，我这里我们要删除用户，那么在删除用户时也需要删除他的用水记录，所以选择After Delete后边的加号按钮

    1. 在红色箭头处写sql语句，就是你要执行的操作,，我们在删除用户的同时也需要删除用户的用水记录，这里的SQL语句就是删除用水表里面的用户用水记录：

  ![](./水用户指南\Snipaste_2021-01-26_21-50-51.png)

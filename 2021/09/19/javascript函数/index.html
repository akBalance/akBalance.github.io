函数

在js调用函数时传递变量参数时，是值传递(极其重要)，但值分两种：普通值，地址值

声明方式

参数

默认参数

在JavaScript中，函数参数的默认值是undefined。意思是当函数需要有参数时，如果没有参数传递给函数，那么参数值默认为undefined。

剩余参数

当参数数量不确定时，参数可以表示为数组。

参数传递

js中的函数参数传递，不管传递的是基本数据类型还是对象，都是值传递，而不是引用传递。在按值传递参数时，值会被复制到一个局部变量。

当传递的对象时，虽然在函数内部修改了对象的属性会反映在外部对象上，但本质还是值传递。传递参数时，值会被复制到一个局部变量tempObj

下面这个例子完全可以说明js中函数的参数传递是值传递,tempObj先去修改name属性，然后指向一个新的对象，在函数执行完成后tempObj被销毁。

原始参数（比如一个具体的数字）被作为形参传递给函数，值不会改变。传递一个对象作为参数，而函数改变了这个对象的属性，这样的改变对函数外部是可见的

arguments对象

函数的实际参数会被保存在一个类似数组的arguments对象中。

箭头函数

提出的原因：更简洁的函数和this的指向问题

预定义函数

javascript中有一些预定义好的函数:eval(), uneval(), isFinite(), isNaN(), parseInt(), parseFloat(), decodeURI()等。。

函数提升

对于函数来说，只有函数声明会被提升到顶部，而函数表达式不会被提升。

函数提升仅适用于函数声明，而不适用于函数表达式

回调函数

1.自定义的 2. 并未调用 3.它自己执行

IIFE

全称：Immediately-Invoked Function Expression

作用：1. 隐藏代码实现 2.不会污染全局命名空间 3.一般用来编写JS模块

this

在函数外部，this代表window。

任何函数本质上都是通过某个对象来调用的，如果没有指定就是Window

所有函数内部都有一个变量this,它的值是调用函数的当前对象（例外：箭头函数是找离自己最近的对象）

闭包

如何产生闭包

当一个嵌套的内部子函数引用了嵌套的外部父函数的变量（或者函数）时，就产生了闭包

闭包到底是什么？

  1. 闭包是嵌套的内部函数
  2. 包含被引用变量(函数)的对象

可以通过F12调试看见

出于种种原因，我们有时候需要得到函数内的局部变量。所以我们在函数的内部，再定义一个函数。并把这个函数作为返回值返回，然后拿到这个函数，我们也就拿到了函数内部的局部变量。

所以，闭包就是能够读取其他函数内部变量的函数。

闭包的用途

  1. 使用函数内部的变量在函数执行后，仍然存活在内存中
  2. 让函数外部可以操作函数内部的数据
  3. 让数据只读，不能写

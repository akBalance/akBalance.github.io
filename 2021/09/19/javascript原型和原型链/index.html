原型

基本数据类型的原型对象

JavaScript 只有一种结构：对象。如自定义的函数是对象，内置的对象Array,String等也是对象。

javascript七种的基本数据类型：Boolean, null, undefined, Number, BigInt, String, Symbol和一种特殊数据类型：Object，都是对象

在js里，everything is object。js大致有两种obejct，一种是function定义的object，这类object有prototype属性，具备造别的object的能力。另一种是var定义的普通object，这类object没有prototype，不具备造对象的能力，回顾Array和Object，很明显，他们是前者

原型基础

实例对象

用new关键字，声明的对象为实例对象。每个实例对象（object）都有一个私有属性（称之为 proto ）指向它的构造函数的原型对象（prototype）。而且每个实例对象没有prototype属性

原型对象

简单来说就是构造器，即构造函数，因为函数也是对象，所以称为原型对象。只有原型对象才有prototype属性

所以实例对象的__proto__等于原型对象的prototype

原型链的属性问题

在原型链中，原型对象的属性和方法可以继承给实例对象,所以我们一般给原型对象添加属性和方法，然后实例对象就可以使用这些方法。

当然，我们也可以在实例对象中重写这些方法。这和java等面向对象语言中的重写子类方法相似。

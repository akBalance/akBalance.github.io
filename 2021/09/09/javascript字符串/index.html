String

声明

一般最好用String字面值的声明方式。这种声明方式也可以使用String对象的任何方法，JavaScript自动把String字面值转换为一个临时的String对象, 然后调用其相应方法,最后丢弃此临时对象。

常用方法

eval函数

eval() 是全局对象的一个函数属性。eval() 的参数是一个字符串。如果字符串表示的是表达式，eval() 会对表达式进行求值。如果参数表示一个或多个 JavaScript 语句，那么eval() 就会执行这些语句。

标签

  1. 行内标签：不独占一行，不能设置宽高，但margin、padding有用。例如：span、label、img、input等。注意 类似 img input 这样的标签被称为可替换标签，它们是可以设置宽高的

  2. 块级标签：独占一行，能设置宽高，例如：div、p等

盒模型

标准盒模型

这种盒子模型，指定元素的宽度和高度，不包括padding和border。真正展示在页面上的盒子大小要加上padding和border，所以输出的coffsetWidth=100+20+20+1+1=142px

怪异盒模型(IE盒模型)

这种盒子模型，包括padding和border。所以展示盒子总体大小还是100px，输出的offsetWidth=100px

felx弹性盒模型

去看看博客，很简单，移动端经常用。

几种宽高的区别

  1. clientWidth, clientHeight：真实设置的宽高+padding

  2. clientTop, clientLeft：指代元素边框的高度

  3. offsetWidth, offsetHeight：真实设置的宽高+padding+border宽度

  4. scrollWidth, scrollHeigh：可视的宽高+隐藏的宽高
